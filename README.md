A terminal-based inventory management system designed to simulate basic stock operations.  
This project focuses on data organization, sequential processing, and state management within a Command Line Interface (CLI) environment.

---

## 🛠️ Technical Stack

- Language: Python 3.x
- Core Logic: Lists, Dictionaries, While-loops, and Conditional Branching
- Interface: Command Line Interface (CLI)

---

## 🚀 Key Features

- Product Registration: Add products with quantity and price.
- Automatic Stock Calculation: Calculates the total value of each product.
- Inventory Summary: Displays a complete overview of all items stored.
- Input Handling: Prevents execution errors using basic exception handling.

---

## 🧠 Learning Objectives

This repository serves as a practical implementation of Computer Science fundamentals, specifically:

1. Data Structures – Using lists and dictionaries to organize inventory data.
2. Control Flow – Managing user interaction with loops and conditional logic.
3. Error Handling – Preventing invalid inputs using Python exceptions.
4. Code Organization – Structuring programs using functions for better readability and maintainability.
