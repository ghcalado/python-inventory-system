# 📦 Inventory Control System | Python Core

A terminal-based inventory management system designed to simulate basic stock operations.  
This project focuses on data organization, sequential processing, and state management within a CLI environment.

---

## 🛠️ Technical Stack

- **Language:** Python 3.x
- **Core Logic:** Lists, Dictionaries, While-loops, and Conditional Branching
- **Interface:** Command Line Interface (CLI)

---

## 🚀 Key Features

- **Product Registration:** Allows users to register products with quantity and price.
- **Automatic Stock Calculation:** Calculates the total value of each product in stock.
- **Inventory Summary:** Displays a complete overview of all products and total stock value.
- **Input Handling:** Basic error handling using try/except to prevent program crashes.

---

## 🧠 Learning Objectives

This repository serves as a practical implementation of Computer Science fundamentals, specifically:

1. **Data Structures:** Using lists and dictionaries to organize inventory data.
2. **Control Flow:** Implementing loops and conditionals to manage user interaction.
3. **Error Handling:** Preventing invalid input using Python exception handling.
4. **Code Organization:** Structuring programs using functions for readability and maintainability.
